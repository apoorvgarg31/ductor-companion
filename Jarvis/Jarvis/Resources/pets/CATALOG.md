# Jarvis Pet Catalog

Bundled sprite atlases (hatch-pet skill output) shipped with Ductor Companion.

Atlas contract: 1536x1872 px, 8 columns x 9 rows, 192x208 cells.

## Pets

### Zen Robot (`zen-robot`)
- Description: Small zen AI robot with a calm posture, pastel tones, and a gentle single-antenna expression.
- Rows in `pet.json`: idle, running-right, running-left, waving, jumping, failed, waiting, running, review
- Atlas size: 1536x1872
- Default sprite for new agents. Users can override via Settings → Agents → Sprite path to point at any `~/.codex/pets/<name>/` they've generated with the hatch-pet skill in Codex CLI.
